## What the project does
This project is a console based Restaurant Ordering System written in C++.  
It allows users to:
- View a restaurant menu with items and prices
- Select items and enter quantities
- Add multiple items to an order
- Generate a bill receipt
- Calculate subtotal automatically
- Apply discounts based on total amount
- Add service charges
- Display the final payable amount

The program uses arrays, loops, functions, and basic input/output operations.

---

## Why the project is useful
This project is useful for:
- Understanding real world use of C++ programming
- Practicing arrays, loops, functions, and conditionals
- Learning how billing systems work
- Building a foundation for larger systems like POS or inventory software

It is especially helpful for beginners and students learning C++.
